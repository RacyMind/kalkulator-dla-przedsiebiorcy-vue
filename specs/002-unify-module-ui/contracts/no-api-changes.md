# Contract: No API Changes

## Zakres

Ten feature dotyczy wyłącznie ujednolicenia UI (pola formularzy i przyciski) w istniejących modułach.

## Zasady

- Brak zmian w endpointach aplikacji.
- Brak zmian w kontraktach danych.

## Zewnętrzne zależności

- Moduł Kontakt korzysta z istniejącego endpointu `contact.php` (bez zmian kontraktu).

## Wymagania testowe

- Istniejące testy jednostkowe logiki modułów muszą przechodzić bez zmian.
