# Implementation Plan: Landing Page SEO

**Branch**: `001-landing-seo` | **Date**: 2026-02-08 | **Spec**: [spec.md](./spec.md)
**Input**: Feature specification from `/specs/001-landing-seo/spec.md`

## Summary

Znaczna poprawa SEO landing page (`landing-page/`) przez:

1. Utworzenie 6 dedykowanych statycznych podstron HTML per kalkulator z rozbudowaną treścią SEO
2. Dodanie `robots.txt` i `sitemap.xml`
3. Rozszerzenie danych strukturalnych (WebSite, BreadcrumbList, HowTo)
4. Wewnętrzne linkowanie między podstronami i rozbudowa footera

Aplikacja SPA (`/app/`) pozostaje bez zmian.

## Technical Context

**Typ projektu**: Statyczny landing page (HTML + Tailwind CSS)
**Hosting**: Apache (`.htaccess` obecny)
**CSS Framework**: Tailwind CSS 4.x (import `@import "tailwindcss"`)
**Czcionki**: Google Fonts — Roboto (400, 500, 700)
**Build**: Tailwind CLI → `dist/style.css`
**Język treści**: Polski
**Docelowa domena**: `https://kalkulatorfinansowy.app/`
**SPA (poza scope)**: Quasar/Vue SPA pod `/app/`, hash routing (`/#/`)

**Istniejąca struktura landing-page/**:

- `index.html` — strona główna
- `style.css` — źródłowy CSS (Tailwind)
- `dist/style.css` — skompilowany CSS
- `tailwind.config.js` — konfiguracja Tailwind
- `.htaccess` — cache headers
- `images/` — grafiki (WebP + PNG fallback)
- `contact.php` — formularz kontaktowy
- `ads.txt` — reklamy

**Key Dependencies**:

- Tailwind CSS 4.x — stylowanie
- Google Fonts (Roboto) — typografia
- Google Tag Manager (GTM-MKR8Z54) — analytics
- Istniejące JSON-LD: SoftwareApplication + FAQPage

## Pre-Implementation Checklist

- [ ] Podstrony używają tego samego layoutu co strona główna (nav, footer, Tailwind, Roboto)
- [ ] Każda podstrona ma unikalny title, description, H1, canonical URL
- [ ] Każda podstrona zawiera min. 300 słów treści SEO
- [ ] Każda podstrona ma JSON-LD BreadcrumbList
- [ ] Każda podstrona ma CTA linkujący do SPA (`/app/#/...`)
- [ ] Każda podstrona ma sekcję „Powiązane kalkulatory"
- [ ] Strona główna zawiera linki do podstron (np. „Dowiedz się więcej")
- [ ] Strona główna zawiera JSON-LD WebSite z SearchAction
- [ ] Footer na każdej stronie zawiera mapę strony
- [ ] `robots.txt` i `sitemap.xml` poprawne
- [ ] Tailwind config obejmuje nowe podstrony
- [ ] Treść w języku polskim
- [ ] Tagi Open Graph i Twitter Cards na każdej podstronie
- [ ] Brak modyfikacji SPA (`/app/`)
